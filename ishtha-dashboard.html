Technology stack — SCADA · IoT · AI Dashboard
⚙️
SCADA — Ignition by Inductive Automation
PLC-connected sensors across fermenters, distillation, utilities. Real-time tag logging every 60 sec. Historian DB stores 2 years of process data. Web-based — MD can view from phone anywhere. India-licensed. Preferred over Wonderware for cost & flexibility.
₹2–4L one-time setup
📡
IoT Sensor Layer — Endress+Hauser / Yokogawa
Vibration transmitters on all rotating equipment (pumps, compressors, centrifuges). Temperature RTDs in fermenters & distillation columns. Ultrasonic level sensors in silos. Coriolis flow meters on ethanol output line. Protocol: Modbus TCP → OPC-UA → SCADA.
Critical instruments: E+H or Yokogawa
📊
AI Dashboard — Power BI + Python ML Pipeline
Power BI pulls from SCADA Historian via API. Python script for anomaly detection — flags abnormal equipment patterns 4–6 hrs before failure. Auto-report PDF generated at 5pm, sent to MD via email + Twilio WhatsApp. Cloud mirror: AWS IoT Core or Azure IoT Hub.
Power BI Pro: ₹800/user/mo
✅
Task Tracker — Zoho Projects + Zapier
Daily checklists auto-created at 6am via Zoho Cliq automation. SCADA-triggered tasks auto-complete when sensor confirms action. Overdue tasks trigger WhatsApp alert to HOD + MD via Zapier webhook. Integrates with all department SOPs in this playbook.
~₹2,000/month total
🔧
CMMS — UpKeep or eMaint
Maintenance work orders, PM schedules, spare parts inventory. IoT vibration alerts auto-create work orders when threshold exceeded. Maintenance Techs close orders on mobile app. MD sees equipment health in Power BI dashboard. Full audit trail for insurance & compliance.
UpKeep Starter: Free tier available
